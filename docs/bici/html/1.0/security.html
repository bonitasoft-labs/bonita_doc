Security

REST APIs

REST APIs are not yet secured. In production only Bonita platform and the plaftorm administrator should be able to access ICI backend by e.g. configuring your network this way.

Elasticsearch

Elasticsearch access should be restricted to the ICI backend. Alternativly you can use Elasticsearch editions that contains security features see Elasticsearch documentation.

REST API extensions

REST API extensions should be secured using Bonita REST API authorization mechanism see Bonita platform documentation.
